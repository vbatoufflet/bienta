```html
<b-toolbar>
    <b-button icon-name="cat" />
</b-toolbar>
```

| Name | Type                                       | Default | Description                                         |
|------|--------------------------------------------|---------|-----------------------------------------------------|
| app  | `boolean`                                  |         | Application toolbar flag.                           |
| clip | `"bottom"`, `"left"`, `"right"` or `"top"` |         | Toolbar clip mode (positioning relative to parent). |
