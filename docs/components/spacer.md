# Spacer

The `<b-spacer>` component serves as spacer while using
[Flexible Box Layout][0]. Its base element having `flex-grow: 1` style applied
automatically.

<b-toolbar>
    <b-label>A</b-label>
    <b-spacer />
    <b-label>B</b-label>
</b-toolbar>

```html
<b-toolbar>
    <b-label>A</b-label>
    <b-spacer />
    <b-label>B</b-label>
</b-toolbar>
```
