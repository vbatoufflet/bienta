## Properties

| Name      | Type                    | Default    | Description                                       |
|-----------|-------------------------|------------|---------------------------------------------------|
| clip      | `"left"` or `"right"`   | `"left"`   | Sidebar clip mode (positioning relative to page). |
| mode      | `"float"` or `"static"` | `"static"` | Sidebar mode.                                     |
| resizable | `boolean`               | `false`    | Sidebar resizable flag.                           |
